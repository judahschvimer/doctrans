eventual consistency
De Consistencia eventual

A property of a distributed system that allows changes to the system to propagate gradually. In a database system, this means that readable members are not required to reflect the latest writes at all times. In MongoDB, reads to a primary have :term:`strict consistency`; reads to secondaries have *eventual consistency*.
A propiedad de un distribuye sistema que permite cambios a el sistema a propagate gradually. En una base de datos system, esto significa que legible miembros no son obligatorias para reflejar el latest escribe at todos times. En MongoDB, lee a un tener primaria :term:`strict consistency`; lee a secondaries tener *eventual consistency*.
